# 📚 EduModule

EduModule adalah aplikasi berbasis Laravel untuk mengelola modul pembelajaran di Universitas Pamulang.

Project ini dibuat sebagai tugas UAS Semester 3 Program Studi Sistem Informasi.

---

## ✨ Fitur

- 🔐 Login Administrator
- 📚 CRUD Data Modul
- 🖼 Upload Cover Modul
- 📄 Upload File PDF
- 👀 Detail Modul
- ⬇ Download PDF
- 📑 Export Data Modul ke PDF
- 📊 Dashboard Statistik
- ✅ Validasi Form
- 🎨 Responsive Bootstrap 5

---

## 🛠 Teknologi

- Laravel 12/13
- PHP 8.x
- MySQL
- Bootstrap 5
- SweetAlert2
- DataTables
- DomPDF

## ⚙ Instalasi

Clone repository

```bash
git clone https://github.com/naldieyy-dotcom/EduModule-UAS.git
```

Masuk ke folder project

```bash
cd EduModule-UAS
```

Install dependency

```bash
composer install
```

Copy file environment

```bash
cp .env.example .env
```

Generate key

```bash
php artisan key:generate
```

Migrasi database

```bash
php artisan migrate
```

Jalankan server

```bash
php artisan serve
```
